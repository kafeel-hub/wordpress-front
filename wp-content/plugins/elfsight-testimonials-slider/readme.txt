== Installation ==

**How to Add Testimonials Slider to WordPress - Automatic Installation (Recommended)**

1. Go to `Plugins > Add New`
2. Search for "Elfsight Testimonials Slider" and install the plugin
3. Activate your Testimonials Slider

**Manual Installation**

1. Download `elfsight-testimonials-slider.1.0.1.zip` from `wordpress.org` and unzip it
2. Upload the `elfsight-testimonials-slider` folder to `/wp-content/plugins/`
3. Go to `Plugins > Installed Plugins` and activate the newest Testimonials Slider

= Manage Elfsight Testimonials Slider =

After installation there are only few steps left before starting to use the plugin and to embed Testimonials Slider to your website.

1. Click 'Testimonials Slider' in WordPress menu to go to Testimonials Slider page.
2. Use the editor to create and setup your first widget.
3. Copy-paste the widget's installation code into any place of your website.
